# Project Title

A postdoc programming challenge: distribute a list of files to a list of nodes.

## Getting Started

Run as 

```
python distribution.py -f files.txt -n nodes.txt
```
for help

```
python distribution -h 
```

### Prerequisites

python2
numpy
matplotlib
pandas
tabulate
